# DAX Measures

This file documents the main DAX measures used in the Power BI Sales Analysis Dashboard.

---

## 1. Core KPI Measures

### Total Quantity

```DAX
Total Quantity = 
SUM(fact_sales[Quantity])
```

**Description:**  
Calculates the total quantity of products sold.

---

### Total Revenue

```DAX
Total Revenue = 
SUM(fact_sales[actual_revenue])
```

**Description:**  
Calculates total actual revenue after discount.

---

### Total Orders

```DAX
Total Orders = 
DISTINCTCOUNT(fact_sales[TransactionNumber])
```

**Description:**  
Calculates the total number of unique transactions/orders.

---

### Total Customers

```DAX
Total Customers = 
DISTINCTCOUNT(fact_sales[CustomerID])
```

**Description:**  
Calculates the total number of unique customers.

---

### AOV

```DAX
AOV = 
DIVIDE(
    [Total Revenue],
    [Total Orders]
)
```

**Description:**  
Calculates the average revenue generated per order.

---

### Basket Size

```DAX
Basket Size = 
DIVIDE(
    [Total Quantity],
    [Total Orders]
)
```

**Description:**  
Calculates the average number of products purchased per order.

---

## 2. Revenue Trend Measures

### Previous Month Revenue

```DAX
Previous Month Revenue = 
CALCULATE(
    [Total Revenue],
    DATEADD(dim_date[sales_date], -1, MONTH)
)
```

**Description:**  
Calculates revenue from the previous month.

---

### Revenue MoM Change

```DAX
Revenue MoM Change = 
[Total Revenue] - [Previous Month Revenue]
```

**Description:**  
Calculates the absolute month-over-month revenue change.

---

### Revenue MoM %

```DAX
Revenue MoM % = 
DIVIDE(
    [Revenue MoM Change],
    [Previous Month Revenue]
)
```

**Description:**  
Calculates the month-over-month revenue growth percentage.

---

## 3. Product Performance Measures

### Total Products

```DAX
Total Products = 
DISTINCTCOUNT(dim_product[ProductID])
```

**Description:**  
Calculates the total number of unique products.

---

### Avg Revenue per Product

```DAX
Avg Revenue per Product = 
DIVIDE(
    [Total Revenue],
    [Total Products]
)
```

**Description:**  
Calculates the average revenue generated per product.

---

### Product Rank

```DAX
Product Rank = 
RANKX(
    ALLSELECTED(dim_product[ProductName]),
    [Total Revenue],
    ,
    DESC,
    DENSE
)
```

**Description:**  
Ranks products by total revenue in descending order.

---

## 4. Customer Behavior Measures

### Repeat Customers

```DAX
Repeat Customers = 
COUNTROWS(
    FILTER(
        VALUES(fact_sales[CustomerID]),
        CALCULATE(DISTINCTCOUNT(fact_sales[TransactionNumber])) > 1
    )
)
```

**Description:**  
Counts customers who made more than one transaction.

---

### Repeat Rate %

```DAX
Repeat Rate % = 
DIVIDE(
    [Repeat Customers],
    [Total Customers]
)
```

**Description:**  
Calculates the percentage of repeat customers.

---

### Customer Basket Size

```DAX
Customer Basket Size = 
DIVIDE(
    'Customer Segmentation'[Customer Quantity],
    'Customer Segmentation'[Purchase Frequency]
)
```

**Description:**  
Calculates basket size at the customer level using customer quantity and purchase frequency.

---

### Avg Customer Basket Size

```DAX
Avg Customer Basket Size = 
AVERAGE('Customer Segmentation'[Customer Basket Size])
```

**Description:**  
Calculates the average basket size across customer segments.

---

## 5. Salesperson Performance Measures

### Total Salespersons

```DAX
Total Salespersons = 
DISTINCTCOUNT(fact_sales[SalesPersonID])
```

**Description:**  
Calculates the total number of unique salespersons.

---

### Avg Revenue per Salesperson

```DAX
Avg Revenue per Salesperson = 
DIVIDE(
    [Total Revenue],
    [Total Salespersons]
)
```

**Description:**  
Calculates the average revenue generated by each salesperson.

---

### Avg Orders per Salesperson

```DAX
Avg Orders per Salesperson = 
DIVIDE(
    [Total Orders],
    [Total Salespersons]
)
```

**Description:**  
Calculates the average number of orders handled by each salesperson.

---

### Salesperson Rank

```DAX
Salesperson Rank = 
RANKX(
    ALL(fact_sales[SalesPersonID]),
    [Total Revenue],
    ,
    DESC,
    DENSE
)
```

**Description:**  
Ranks salespersons by total revenue in descending order.

---

### Salesperson Revenue Share

```DAX
Salesperson Revenue Share = 
DIVIDE(
    [Total Revenue],
    CALCULATE(
        [Total Revenue],
        ALL(fact_sales[SalesPersonID])
    )
)
```

**Description:**  
Calculates each salesperson's revenue contribution share.
